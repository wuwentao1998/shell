# Shell

This repository contains a bash-like command line interface, which supports features like pipe, redirection and background jobs.

## Envoirment
- OS: Ubuntu 18.04.1

- Compiler: clang 6.0.0

## Build
``` bash
cd ./code

cmake -DCMAKE_C_COMPILER=clang

make
```

## Usage
``` bash
./mumsh
```
